# Hyper

A simple web UI to chat with different LLMs.

## Setup

1.  **Install Dependencies:**
    ```
    pip install -r requirements.txt
    ```

2.  **Set API Keys:**
    Create a `.env` file in the main directory and add your API key:
    ```
    GEMINI_KEY=<Your Google Gemini API Key>
    ```

## Run

On Windows, just double-click `run.bat`.

This will start the server and open the app in your browser at `http://127.0.0.1:8000`.
